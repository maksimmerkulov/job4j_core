Professional implementation of Java Core: algorithms, data structures, OOP, and functional programming (Stream API, Lambda). Developed as part of the Job4j training program.

## Overview
This repository serves as a comprehensive collection of solutions, starting from basic syntax and evolving into complex architectural patterns and data structures. The project emphasizes the transition from procedural to functional programming, with a focus on code efficiency and maintainability.

### Key Concepts Covered
* **Java Fundamentals**: Deep dive into primitive types, control flow, and memory management.
* **OOP Principles**: Implementation of classes, objects, and nested structures to build modular applications.
* **Data Structures**: Practical experience with List, Set, Map, and Tree implementations.
* **Functional Programming**: Modern Java development using Lambda expressions and the Stream API.
* **Algorithmic Thinking**: Problem-solving using advanced techniques like the Two Pointers approach.

## Engineering Standards
The project follows industry-standard development practices:
* **Clean Code**: Readable and maintainable code structure adhering to Java naming conventions.
* **Maven Integration**: Automated lifecycle management for building and dependency resolution.
* **Algorithmic Optimization**: Focus on time and space complexity ($O(n)$ analysis) when working with collections.

## Tech Stack
* **Language**: Java 17 (LTS)
* **Build System**: Maven
* **Testing**: JUnit 5, AssertJ
